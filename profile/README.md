# ANSILITHIC

A collection of command-line tools designed for macOS.

---

### Tools

| Tool | Description |
|------|-------------|
| [**ping**](https://github.com/ansilithic/ping) | Enhanced ping with timestamps, color-coded output, and summary stats |
| [**rm**](https://github.com/ansilithic/rm) | Drop-in `/bin/rm` replacement that moves files to Trash |
| [**rmdir**](https://github.com/ansilithic/rmdir) | Drop-in `/usr/bin/rmdir` replacement that moves directories to Trash |
| [**swift-cli-core**](https://github.com/ansilithic/swift-cli-core) | Shared library — terminal styling, table formatting, structured output |
