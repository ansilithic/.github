# ANSILITHIC

A collection of command-line tools designed for macOS.

```sh
brew tap ansilithic/tap
```

---

### Tools

| Tool | Install | Description |
|------|---------|-------------|
| [**ping**](https://github.com/ansilithic/ping) | `brew install ansilithic/tap/ping` | Enhanced ping with timestamps, color-coded output, and summary stats |
| [**rm**](https://github.com/ansilithic/rm) | `brew install ansilithic/tap/rm` | Drop-in `/bin/rm` replacement that moves files to Trash |
| [**rmdir**](https://github.com/ansilithic/rmdir) | `brew install ansilithic/tap/rmdir` | Drop-in `/usr/bin/rmdir` replacement that moves directories to Trash |

### Libraries

| Library | Description |
|---------|-------------|
| [**swift-cli-core**](https://github.com/ansilithic/swift-cli-core) | Terminal styling, table formatting, and structured output for Swift CLIs |
